# 🎮 HTML Tetris Game

HTML, CSS, JavaScript만 사용하여 만든 **브라우저 테트리스 게임**입니다.  
별도의 라이브러리나 빌드 과정 없이 **단일 HTML 파일**로 실행할 수 있습니다.

---

## ✨ 주요 기능

- 🧱 테트리스 기본 블록 (I, O, T, S, Z, J, L)
- ⏭️ 다음 블록 미리보기 (Next Piece)
- 🎯 점수 시스템
- 💀 게임 오버 화면
- 🔁 **Enter 키로 즉시 재시작**
- ⚡ **Space 키 하드 드롭 (즉시 낙하)**
- ⌨️ 키보드 조작
- 📦 단일 HTML 파일 구조
- 🎨 Canvas 기반 렌더링

---

## 🎮 조작 방법

| 키 | 동작 |
|----|------|
| ⬅️ | 왼쪽 이동 |
| ➡️ | 오른쪽 이동 |
| ⬇️ | 한 칸 내려가기 |
| ⬆️ | 블록 회전 |
| Space | 하드 드롭 (즉시 낙하) |
| Enter | 게임 오버 후 재시작 |
